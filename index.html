Hoi!
Ik ben Isa en volg de opleiding Communication & Multimedia design aan de Hogeschool van Amsterdam. Via deze website die ik zelf heb gecodeerd met HTML/CSS neem ik jullie mee door een aantal projecten die ik heb gedaan tijdens mijn studie.

Skills

Bij de opleiding CMD wordt je opgeleid op verschillende gebieden. Ik heb hierdoor kennis op het gebied van Interaction Design, Front-end Development en Visual Interface Design. Mijn focus ligt vooral op Interaction Design en Visual Interface Design maar vind het ook nog steeds interessant om mij verder te ontwikkelen op het gebied van Front-end Development.

Projecten

Project Responsive Multi Device Design

Een ontwerp voor een interactieve ervaring waarbij de oplossing specifiek het gebruik van verschillende devices omvat.

De gebruiker kan hierbij een wandeling samenstellen in een stad waar hij/zij nog niet eerder is geweest. Tijdens de wandeling maakt de gebruiker gebruik van zijn telefoon en/of smartwatch.

Vormgeving

Een Responsive one page website voor het evenement ‘Plop de dag’ waarop alle informatie te lezen is.

De website is ontworpen binnen de huisstijl van de opdrachtgever, de gemeente Rotterdam. Allereerst is de huisstijl geanalyseerd om dit vervolgens te gebruiken in het ontwerpen van verschillende schermen.

Frontend development

2 verschillende pagina’s van een website uitgewerkt met ‘nette’ vanilla HTML, CSS en Javascript.

De gekozen website is van het merk Maha. Hierbij heb ik extra aandacht besteed aan de surface plane. Dit heb ik gedaan door gebruik te maken van custom properties, animaties, micro-interacties en advanced positioning.

Webapplicaties

Een website voor het merk Stylish! Ik heb hiervoor verschillende onderdelen van drie webapplicaties onderzocht en geïnstalleerd.

Ik heb de website gemaakt via Wordpress en afgestemd op de wensen van de opdrachtgever. Zo was het belangrijk dat klantenservice voorop stond en nieuwe kledingmerken zich gemakkelijk konden aanmelden.

Visual interface design

Een interface ontwerp voor een middelbare school waarmee er in de school bibliotheek op eenvoudige wijze boeken kan worden gezocht.

Ik heb gewerkt met de huisstijl van de Gemeente Amsterdam maar hier binnen een eigen substijl ontwikkelt die de doelgroep meer aanspreekt.

Project beyond

Het verminderen van een irritatie tussen huisgenoten. Hierbij ligt de focus niet op beeldschermen maar op de interactie met computers en sensoren die verwerkt zijn in de objecten.

De gekozen design challange is "Hoe kunnen we doormiddel van een digitaal interactieve oplossing ervoor zorgen, dat de irritatie van een te lang zittende volle afvalbak en de bijhorende afval eromheen wordt voorkomen?”.

Inleiding programmeren

Eerste keer werken met JavaScript en hiervoor de basis leren. Keuze uit verschillende soorten opdrachten.

Ik heb hiervoor een website gemaakt waarbij het mogelijk is je eigen pizza samen te stellen.

User centred design

Een advies over hoe de ervaring kan worden verbeterd die gebruikers hebben met de producten van de opdrachtgever ‘Maas International’.

De opdracht bestaat uit het uitvoeren van een gebruikersonderzoek naar de doelgroep. Dit advies breng je over in de vorm van een posterpresentatie en productbiografie ter ondersteuning.

Informatie architectuur

Voor het platform A Tavola een informatie architectuur opgesteld en hiervoor verschillende relevante pagina's ontworpen.

De applicatie helpt de gebruikers bij het maken van reserveringen bij verschillende horeca gelegenheden.
